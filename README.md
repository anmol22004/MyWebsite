# My Site — Vite + React + TypeScript + Tailwind

Starter scaffold.

Quick start:

```bash
# from project root
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

Notes:
- Add `react-router-dom` for routing: `npm install react-router-dom`
- Add MDX support with `vite-plugin-mdx` if you want a blog.
- Deploy to Vercel, Netlify, or Cloudflare Pages.
